Java Web Application (Apache Tomcat)

Type: Java-based web application

Tech: Java, HTML, XML, Apache Tomcat

Overview

This project was a Java-based web application developed as part of a university coursework assignment. The application was deployed using Apache Tomcat and integrated frontend components built with HTML. XML configuration files were used to define application structure and server behavior. The project demonstrated core web development concepts, including server-side processing, application deployment, and request handling.

Key Contributions

  • Developed server-side functionality using Java.
  • Built frontend pages using HTML for user interaction.
  • Configured application behavior using XML configuration files.
  • Deployed and tested the application on an Apache Tomcat server.
  • Debugged server and application errors during development.

What I Learned

  • How Java web applications are structured and deployed.
  • How Apache Tomcat handles web applications and requests.
  • The role of XML in configuring Java web applications.
  • How frontend and backend components interact in a web environment.
  • How to troubleshoot server-side and deployment issues.

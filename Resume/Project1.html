Somnium

Type: 2D top down video game

Tech: C#, Unity, GitHub

Overview

Somnium is a 2D top down video game built in Unity using C#. The game features a player navigating through various levels, overcoming obstacles, and collecting items to progress. The project was developed as part of a coursework assignment to demonstrate proficiency in game development concepts, including physics, input handling, and scene management.

Key Contributions

  • Created various scripts to support different functions in the game.
  • Tested and validated code before pushing changed.
  • Managed audio in the the video game.
  • Assisted with implementing an AI system.
  • Worked with team members to do code review and push changes.

What I Learned

  • How to work in Unity and create a video game.
  • I learned how to manage a big coding project in a group through version control.
  • I learned a lot about Git and how branching, merging, pushing, and pulling works.
  • I learned how to code in C#.
